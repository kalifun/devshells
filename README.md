# devshells
custom nix devshells

# usage

add this line to your `.envrc` file:
```text
use flake github:acehinnnqru/devshells#go-latest
```

examples:
```text
use flake github:acehinnnqru/devshells#rust-stable
use flake github:acehinnnqru/devshells#python-313
use flake github:acehinnnqru/devshells#nodejs-22
```
